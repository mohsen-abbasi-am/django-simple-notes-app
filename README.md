# 📝 Django Simple Notes App

A minimalist web application to create, edit, and delete notes — built with Django & Bootstrap 5.

## ✨ Features

- 🆕 Add new notes  
- ✏️ Edit existing notes  
- 🗑️ Delete notes  
- 📋 List all notes  
- 🎨 Clean UI with Bootstrap
